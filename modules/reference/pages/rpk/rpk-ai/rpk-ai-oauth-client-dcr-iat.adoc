= rpk ai oauth-client dcr iat
:description: rpk ai oauth-client dcr iat
:page-preview: true

Manage Initial Access Tokens (IATs) — one-shot bearer credentials a
caller presents to the public registration endpoint when the tenant's
admission mode is initial-access-token. The plaintext is printed
exactly once on mint; only a hash is stored.

== Usage

[,bash]
----
rpk ai oauth-client dcr iat [command]
----
