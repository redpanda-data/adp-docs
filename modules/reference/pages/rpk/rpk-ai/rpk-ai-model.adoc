= rpk ai model
:description: rpk ai model

Discover the models the Redpanda AI gateway exposes.

The catalog is read-only — `model list` and `model get` are the only verbs. The
catalog is populated from each LLM provider's metadata plus any extras the
operator has pinned to a tenant.

Aliases: `models`, `m`.

== Usage

[,bash]
----
rpai model [command]
----

== Aliases

[,bash]
----
model, models, m
----

== Flags

[cols="1m,1a,2a"]
|===
|*Value* |*Type* |*Description*

|-h, --help |- |help for model.

|-o, --format |string |output format: table|wide|json|yaml|markdown (env: RPAI_FORMAT) (default "table").

|--no-color |- |disable colored output (env: NO_COLOR).

|-c, --rpai-config |string |path to rpai config (env: RPAI_CONFIG) (default "/var/lib/redpanda/.rpai/config").

|-s, --rpai-endpoint |string |override dataplane URL (env: RPAI_ENDPOINT).

|-p, --rpai-profile |string |rpai profile name (env: RPAI_PROFILE).

|-v, --rpai-verbose |- |verbose debug logging to stderr (env: RPAI_VERBOSE).

|--token |string |static bearer token override (env: RPAI_TOKEN).
|===
