= rpk ai oauth
:description: rpk ai oauth
:page-beta: true

Manage OAuth authorization-server configurations registered with the Redpanda AI gateway.

OAuth providers describe third-party authorization servers (GitHub, Google, Slack, ...)
that user-facing MCP servers can authenticate against via the device-consent flow.

Aliases: `oauth-provider` (compat with aigwctl), `op`.

== Usage

[,bash]
----
rpai oauth [command]
----

== Aliases

[,bash]
----
oauth, oauth-provider, op
----
