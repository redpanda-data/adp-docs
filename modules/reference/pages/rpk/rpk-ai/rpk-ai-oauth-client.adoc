= rpk ai oauth-client
:description: rpk ai oauth-client
:page-beta: true

Manage OAuth clients registered with the AI gateway's in-dataplane
OAuth Authorization Server.

An OAuthClient is the external tool (Claude.ai, ChatGPT, Cursor, …) that
requests access tokens for an MCP. Create one per tool you want to allow
in; the client_secret is returned exactly once on create and cannot be
retrieved afterward. Rotate via delete + re-create until Phase 2 lands
a dedicated rotation RPC.

Aliases: `oauth-clients`, `oc`.

== Usage

[,bash]
----
rpk ai oauth-client [command]
----

== Aliases

[,bash]

oauth-client, oauth-clients, oc
